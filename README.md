# HowToUse_Git
GitHub 사용법에 대해 Velog에 정리한 걸 GitHub Actions+GraphQL API를 이용한 동기화를 활용해 업로드합니다.


## 구현 기능

- Velog 시리즈별 포스트 동기화
- 마크다운 형식 자동 변환
- 6시간마다 자동 업데이트

<br>

## 구현 방법

- GitHub 저장소 설정
- GitHub Actions 워크플로우 작성
- Python 스크립트 작성
<br>

## 사용 방법

- 초기 설정 방법
- 자동 동기화 확인
- 수동 동기화 방법
<br>

## 추가 커스터마이징

- 다른 시리즈 동기화
- 동기화 주기 변경
- 마크다운 형식 수정
